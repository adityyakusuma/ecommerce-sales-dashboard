# 📊 E-Commerce Sales Analytics Dashboard

Interactive and responsive business analytics dashboard built using HTML, CSS, JavaScript, Chart.js, and CSV-based data processing.

This project was developed as a portfolio project for a Data Analyst / Business Intelligence role to demonstrate data visualization, KPI reporting, customer analytics, and dashboard interactivity.
